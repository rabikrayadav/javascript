evel() function

The eval() function evaluates or executes an argument.
If the argument is an expression, eval() evalutes the expression.
If the argument is one or more javascript statement, eval() executes the statements.

Example:
var x = 10;
var y = 20;
var a = eval("x * y")
var b = eval("2 + 2")
var c = eval("x + 17")
var res = a + b + c;
document.write(a);
document.write(b);
document.write(c);
document.write(res);
