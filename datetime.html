SetInterval() and clearInterval() Method

setInterval()
The setInterval() method calls a function or evaluates an expression at specified intervals (in milliseconds).
This method will continue calling the function until clearInterval() is called, or the window is closed.
The ID value returned by setInterval() is used as the paramenter for the clearInterval() method.
