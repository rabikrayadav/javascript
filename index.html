1) Variable in javascript

JavaScript variable are containers for storing data values.
Creating a variable in javascript is called "declaring" a variable.
For declare a javascript variable we have to write "var" keyword before the variable name.
All javascript variables must be identified with unique names.
These unique name are called identifiers.
var a;
a = 10;
var name = "rabi kr yadav";

Operator in javascript

javascript operators are used to assign values, compare values, perform arithmetic operations, and more.
There are different types of operators for different uses.
Arthmetic operators description
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus (remainder)
++ Increment
-- Decrement
Assignment Operators description
= Assign
+= Add and assign.
For example, x+=y is same as x=x+y.
-= Subtract and assign. For example, x-=y is same as x=x-y.
*= Subtract and assign. For example, x*=y is same as x=x*y.
/= Subtract and assign. For example, x/=y is same as x=x/y.
%= Subtract and assign. For example, x%=y is same as x=x%y.
