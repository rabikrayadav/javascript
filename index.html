1) Variable in javascript

JavaScript variable are containers for storing data values.
Creating a variable in javascript is called "declaring" a variable.
For declare a javascript variable we have to write "var" keyword before the variable name.
All javascript variables must be identified with unique names.
These unique name are called identifiers.
var a;
a = 10;
var name = "rabi kr yadav";

Operator in javascript

javascript operators are used to assign values, compare values, perform arithmetic operations, and more.
There are different types of operators for different uses.
Arthmetic operators description
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus (remainder)
++ Increment
-- Decrement
Assignment Operators description
= Assign
+= Add and assign.
For example, x+=y is same as x=x+y.
-= Subtract and assign. For example, x-=y is same as x=x-y.
*= Subtract and assign. For example, x*=y is same as x=x*y.
/= Subtract and assign. For example, x/=y is same as x=x/y.
%= Subtract and assign. For example, x%=y is same as x=x%y.

Comparison Operator description
== is equal to
=== is indentical (is equal to and is of the same data type)
!= is not equal to
!== is not indentical
> greater than
>= greater than or equal to
< less than
<= less than or equal to

Conditional Statement

Conditional statement are used to perfom different actions based on different conditions.
  • if statement
    • use this statement to execute some code olny if a specified conditon is true.
    Syntax:
    if(condition)
    {
    code to be executed if condition is true
    }
    Example:
    var marks = 100;
    if (marks == 100)
    {
    document.write("Your mark is correct")
    }
    Result:
  • if...else statement
    • use this statement to execute some code if the condition is true and another code if the condition is false
    Syntax:
    if(condition)
    {
    code to be executed if condition is true
    }else
    {
    Another code to be executed if condition is false
    }
    Example:
    var marks = 100;
    if (marks == 200)
    {
    document.write("Your mark is correct")
    }else {
    document.write("You mark is not correct");
    }
    Result:
  • Switch statement
    • statement is used to select one of many blocks of code to be executed.
    Syntax:
    switch(expression)
    {
    case constant : code;
    break;
    case constant : code;
    break;
    default: code;
    }
    Example:
    switch (choice) {
    case "add" : var a = 8;
    var b = 2;
    var sum = a + b;
    document.write("Addition of A and B is " + sum);
    break;
    case "sub" : var a = 8;
    var b = 2;
    var sub = a - b;
    document.write("Subtraction of A and B is " + sub);
    break;
    case "mult" : var a = 8;
    var b = 2;
    var mult = a * b;
    document.write("Multiplication of A and B is " + mult);
    break;
    default: alert("not a valid input");
    break;
    }
