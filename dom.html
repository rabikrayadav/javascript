DOM :- D : Document, O : Object, M : Model

When a web page is loaded, the browser creates a Document Object Model of the page. According to W3C everything is a Node.

documents are modeled using objects, Element and attributes in our page is an Object.

A start tag consists of an opening angle bracket(<) followed by the element name, zero or more space separated attribute/value pairs, and a closing angle bracket(>).

An element in HTML represents some kind of structure or semantics and generally consists of a start tag, content, and an end tag.

An attribute defines a property for an element, consists of an attribute/value pair, and appears within the element's start tag.
